# Arrowgram Editor

## Usage

1. Install [elm](https://guide.elm-lang.org/install.html).
2. In this directory, run `elm reactor`.
3. Go to `http://localhost:8000/src/index.elm`.

## Compilation

1. Install [elm](https://guide.elm-lang.org/install.html).
2. In this directory, run `make`.
3. See the directory `dist/`.
